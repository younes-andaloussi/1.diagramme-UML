# Exercices UML - Enonces et Corrections

## Presentation

Ce depot regroupe l'ensemble des exercices UML realises dans le cadre du cours **Analyse et Conception**.

L'objectif du projet est de preparer l'examen en refaisant chaque exercice a partir de son enonce, sans consulter directement les corriges. Cette methode permet de renforcer la reflexion, la comprehension des concepts UML et l'autonomie dans la modelisation.

Le travail rassemble **14 exercices** couvrant les principaux diagrammes etudies en cours :

- diagrammes de cas d'utilisation ;
- diagrammes de classes ;
- diagrammes de sequence.

## Objectifs pedagogiques

- Comprendre les besoins metier a partir d'un enonce.
- Identifier les acteurs et les fonctionnalites d'un systeme.
- Construire des diagrammes de cas d'utilisation coherents.
- Identifier les classes metier pertinentes.
- Definir correctement les attributs, methodes et associations.
- Utiliser les cardinalites UML.
- Modeliser les interactions entre objets avec des diagrammes de sequence.
- Developper une logique de conception orientee objet.

## Methode de travail

Pour chaque exercice :

1. Lire et analyser completement l'enonce.
2. Realiser les diagrammes sans consulter le corrige.
3. Verifier et corriger les erreurs eventuelles.
4. Comparer avec la solution attendue.
5. Ameliorer la modelisation lorsque c'est necessaire.

Cette approche permet de reproduire les conditions de l'examen et de renforcer les reflexes de conception UML.
